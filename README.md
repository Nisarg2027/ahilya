# 🛡️ Ahilya Rakshasutra

> **🏆 4th Place Winner - Indore Tech Hackathon**

Ahilya Rakshasutra is a comprehensive, AI-powered cybersecurity and threat-detection platform. It features a React-based admin dashboard for monitoring threats, a Flutter mobile application for end-users, and a robust FastAPI backend powered by Machine Learning to detect malware, deepfakes, and spam in real-time.

## 🛠️ Tech Stack
**Frontend (Web):**
* React.js / Vite
* TypeScript
* Tailwind CSS
* Leaflet (Interactive Maps)

**Frontend (Mobile):**
* Flutter / Dart
* HTTP for API integration

**Backend & Machine Learning:**
* Python / FastAPI
* PyTorch (Deepfake Detection)
* Scikit-Learn (Random Forest Malware & Spam Classification)
* Uvicorn
* Hosted on Hugging Face Spaces (Docker / Git LFS)

---

## 🚀 Features
* **Malware Detection:** Upload APKs or files to be scanned against an optimized Random Forest model.
* **Deepfake Analysis:** Utilizes EfficientVit (PyTorch/ONNX) to analyze image authenticity.
* **Spam Filtering:** Classifies SMS and URLs using vectorization and ML models.
* **Live Threat Map:** Visualizes localized cyber threats via an interactive Leaflet map on the admin dashboard.
* **Cross-Platform:** Accessible via the web dashboard for admins and the Android app for users.
